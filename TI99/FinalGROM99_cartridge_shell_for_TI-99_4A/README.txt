# Summary

This is a cartridge shell design for the FinalGROM 99 (https://endlos99.github.io/finalgrom99/) board.  All models are in mm

It is a modification of acadiel's TI-99/4A Standard Cartridge Shell.  It has been modified with an opening for the boards SD slot and buttons for control, an embossed name, and bottom surface of the cart has been flattened for 3D printing friendliness. 

The buttons are designed to work with the boards original installed switches.

Notes: 
I printed mine in ABS at .3 layer height.  The top shell was printed vertically (side nearest the slot down) and the bottom shell horizontally.
